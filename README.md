# نظام غزة كاش المتكامل لإدارة المبيعات والمخازن والحسابات (Gaza Cash ERP) 📊✨

مرحباً بك في **نظام غزة كاش الذكي (Gaza Cash ERP)**، تطبيق سطح المكتب المتكامل والآمن المصمم خصيصاً لتلبية احتياجات التجار، المحلات، والشركات في إدارة الأنشطة التجارية والمخازن والحسابات بكفاءة قصوى ومرونة عالية.

تم بناء النظام بالكامل باستخدام تقنيات حديثة واحترافية: **Electron + React + Vite + TypeScript + Tailwind CSS**، مع تقديم واجهات مستخدم مصممة بدقة وعناية تدعم اللغتين العربية والإنجليزية وتراعي أرقى معايير تجربة المستخدم المالي.

---

## 🌟 مميزات النظام الأساسية

1. **إدارة نقاط البيع الذكية (Interactive POS)**:
   - واجهة كاشير تفاعلية سريعة تدعم الاختصارات السريعة باللوحة المفاتيح.
   - دعم كامل للربط مع أجهزة قارئ الباركود وموازين المنتجات.
   - طباعة فواتير حرارية احترافية مع توليد تلقائي لرموز الاستجابة السريعة (QR Code).

2. **الرقابة وإدارة المخازن والمستودعات (Inventory Control)**:
   - جرد الأصناف، متابعة كميات المنتجات وتحديثها لحظياً.
   - التنبيه التلقائي بالأصناف التي تحت خطر نفاد المخزون.
   - إدارة التحويلات بين الفروع وتتبع حركة الأصناف بالتفصيل.

3. **المبيعات والمشتريات وإدارة المرتجعات (Trading Management)**:
   - نظام متكامل للفواتير والطلبيات وعروض الأسعار.
   - دعم كامل للمرتجعات (مرتجع مبيعات ومرتجع مشتريات).
   - توثيق تفصيلي لكل عملية مع تحديد حالة السداد (نقدي، آجل، دفعات).

4. **إدارة حسابات العملاء والموردين وكشوف الحساب (Relations & Accounts)**:
   - كشوف حسابات تفصيلية لكل عميل ومورد متضمنة الأرصدة والديون والمدفوعات السابقة.
   - جدولة الديون ومتابعة السداد بمرونة لتجنب أزمات السيولة.

5. **المستشار المالي والذكي المدمج (Smart Financial AI Advisor)**:
   - مساعد تحليلي ذكي يقوم بتحليل فوري لأرقام المبيعات، الأرباح، والديون.
   - تقديم توصيات تشغيلية مخصصة لتحسين الأداء وتقليل الخسائر والتحكم في المخازن.

6. **أمن وحماية متطورة وفحص جدار الحماية (Firewall & Backup)**:
   - نظام حماية وجدار حماية مدمج لمراقبة محاولات الدخول وحظر التهديدات.
   - إدارة صلاحيات المستخدمين بدقة (مدير نظام، كاشير).
   - تشفير متقدم للبيانات ونسخ احتياطي آمن بضغطة زر واحدة.

## 🛠️ متطلبات بيئة المطورين والتشغيل

* **Node.js**: إصدار `18.0.0` أو أحدث.
* **npm**: مدير حزم Node.js المدمج.

## 🚀 تشغيل التطبيق في بيئة التطوير (Development)

لتشغيل واجهة النظام في خادم التطوير وتجربة الميزات مباشرة:

1. قم بتثبيت الاعتمادات المطلوبة للمشروع:
   ```bash
   npm install
   ```

2. تشغيل خادم التطوير لواجهات الويب:
   ```bash
   npm run dev
   ```

3. تشغيل التطبيق كبرنامج سطح مكتب محلي عبر Electron:
   ```bash
   npm run electron:dev
   ```

## 📦 البناء واستخراج الملف التنفيذي النهائي (`.exe`) لويندوز

النظام مجهز بالكامل بأداة **`electron-builder`** المؤتمتة لإنتاج ملفات جاهزة للعملاء بضغطة زر واحدة دون متطلبات تشغيل مسبقة:

* **توليد النسخة المحمولة (Portable) وإصدار التثبيت الرسومي (NSIS Installer) معاً**:
  ```bash
  npm run build:win
  ```

* **توليد النسخة المحمولة (Portable) فقط (ملف تشغيل مباشر دون تثبيت)**:
  ```bash
  npm run electron:build:portable
  ```

* **توليد نسخة التثبيت الرسومي (NSIS Installer) فقط**:
  ```bash
  npm run electron:build:nsis
  ```

### 📂 أين تجد الملفات المستخرجة؟
بمجرد انتهاء معالج التعبئة، ستجد الملفات التنفيذية داخل مجلد:
📂 **`release/`** في المجلد الرئيسي للمشروع.

## 🎨 تفاصيل إعدادات الأيقونات والتثبيت الرسومي

- **الأيقونات الرسمية**: تم توليد أيقونة عالية الدقة بمقاسات متعددة مدمجة داخل ملف `build/icon.ico` لضمان ظهور شعار التطبيق الحاد في كافة أجزاء نظام التشغيل (سطح المكتب، شريط المهام، قائمة ابدأ).
- **إعدادات معالج التثبيت (NSIS)**:
  - `"oneClick": false` لتزويد المستخدم بمعالج رسومي تفاعلي احترافي أثناء التثبيت.
  - `"allowToChangeInstallationDirectory": true` يتيح للمستخدم تحديد المجلد المخصص لتثبيت البرنامج (مثلاً على قرص D أو C).
  - `"perMachine": true` يتيح خيار تثبيت البرنامج لجميع المستخدمين على الكمبيوتر الشخصي.
